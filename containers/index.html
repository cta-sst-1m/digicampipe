Containers and Data Levels¶

class digicampipe.io.containers.R0CameraContainer(**fields)[source]¶

Storage of raw data from a single telescope

Parameters

  • pixel_flags (ndarray (n_pixels, ) (bool)) – a ndarray to flag the pixels

  • adc_samples (ndarray (n_pixels, n_samples, ) (uint16)) – a ndarray (n_pixels, n_samples) containing the waveforms in each pixel

  • adc_sums (ndarray (n_channels, x n_pixels)) – numpy array containing integrated ADC data (n_channels, x n_pixels)

  • baseline (ndarray (n_pixels, ) (float)) – baseline holder for baseline computation using clocked triggers

  • digicam_baseline (ndarray (n_pixels, ) (uint16)) – baseline computed by DigiCam of pre-samples (using 1024 samples)

  • standard_deviation – baseline standard deviation holder for baseline computed using clocked triggers

  • dark_baseline (ndarray (n_pixels, ) (float)) – baseline holder for baseline computed in dark condition (lid closed)

  • hv_off_baseline (ndarray (n_pixels, ) (float)) – baseline computed with sensors just bellow breakdown voltage (or without bias voltage applied)

  • camera_event_id ((int)) – unique event identification provided by DigiCam

  • camera_event_number ((int)) – event number within the first trigger of operation

  • local_camera_clock ((int)) – time stamp from internal DigiCam clock (ns)

  • gps_time ((int)) – time stamp provided by a precise external clock (synchronized between hardware components)
